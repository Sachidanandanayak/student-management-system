# EduManage

A modern Student Management System built using React, FastAPI, and MySQL.

## Features

- JWT Authentication
- Role-Based Access
- Admin Dashboard
- Teacher Dashboard
- Student Dashboard
- Attendance Management
- Marks & Results
- Fee Management
- Notifications
- Analytics Dashboard
- Responsive UI

## Tech Stack

Frontend:
- React
- TypeScript
- Tailwind CSS
- Framer Motion

Backend:
- FastAPI
- SQLAlchemy
- JWT

Database:
- MySQL

npm run dev
